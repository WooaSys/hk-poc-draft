## 비교

| 기준 | 일평균 | 비고 |
|---|---:|---|
| 실제 (전체 톤수) | 3,969만원 | 3PL 제외 |
| 추정 (3.5T only) | 4,590만원 | VROOM + 거리 보정 |
| **차이** | **+621만원 (+16%)** | |

### 차이 원인

1. **톤수 혼합 효과**: 실제는 1T(저렴) + 5T(비쌈) 혼합이지만 3.5T는 중간 단가 → 과대 추정 가능
2. **거리 보정계수**: 구간별 보정계수가 과대 추정일 수 있음
3. **3PL 미포함**: 실제 비용에서 3PL 3,453건의 비용이 빠져 있음 (포함 시 실제 비용이 더 높아짐)
4. **배차 최적화 차이**: VROOM 경로와 실제 배차 경로가 다름

### 결론

- 3.5T 추정값은 실제 대비 약 +16% 수준
- 3PL 비용 미포함을 고려하면 실제 차이는 더 작을 수 있음
- **시나리오 간 상대 비교 목적으로는 충분한 정확도**
