## 추정 방법

- 차량 번호/기사 ID 데이터 없음 → **일 최대 동시 출발 건수**로 최소 보유 대수 추정

### 편차 해석

- **CV(변동계수)**: 표준편차 ÷ 평균. 낮을수록 일일 운행이 안정적
- **CV 25% 이하**: 안정적 (칠곡 5T 24.8%, 제주 11RB 27.0%)
- **CV 30~50%**: 보통 (주요 트럭 대부분)
- **CV 50% 이상**: 변동 큼 (8T/11T 소형 편성, 3PL, 제천 1T)
- **min=1인 조합이 많음**: 토요일/공휴일 등 저물량일 포함으로 추정
- **max/min 비율**: 대부분 10x 이상 → 피크 대비 저물량일 편차가 매우 큼

## 특이사항

- **평택RDC 3PL 집중**: 일 최대 155건으로 자체 배차(63건)의 2.5배 → 수도권 소량 배송을 대규모 외주
- **2월 → 11월 증가**: 대부분 RDC에서 자체 차량 운행이 증가 (물량 증가 또는 자체 배송 전환)
- **칠곡 3PL 감소**: 2월 37건 → 11월 19건, 자체 배송 전환 가능성
- **차량 번호 없음**: 동일 차량의 RDC 간 이동 여부는 현재 데이터로 확인 불가, 추가 데이터 필요
