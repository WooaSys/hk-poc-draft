# 거리 보정계수 (OSRM 거리 → 실제 배송 거리)

- **작성시간**: 2026-04-03
- **데이터 기준**: 배송 실적 2025년 2월, 11월 (340,040건)

---

## 목적

OSRM으로 계산한 RDC→시군구(시청 대표좌표) 거리는 실제 배송 거리보다 짧다. 시군구 내부 배송지 순회 거리가 빠져 있기 때문. 실제 배송 실적의 `Distance by Shipment`와 비교하여 거리 구간별 보정계수를 산출.

---

## 산출 방법

1. 배송 실적의 `Distance by Shipment` = 실제 배송 거리
2. OSRM `distance_km` = RDC → 시군구(시청) 거리
3. 보정비율 = 실제 거리 / OSRM 거리
4. OSRM 거리 구간별 중앙값 사용

---

## 보정계수 테이블

| OSRM 거리 구간 | 보정계수 | 건수 | 비고 |
|---|---:|---:|---|
| 0~30 km | 1.82 | 51,507 | RDC 근처, 내부 순회 비중 큼 |
| 30~60 km | 1.52 | 75,277 | |
| 60~90 km | 1.25 | 81,903 | |
| 90~120 km | 1.25 | 56,588 | |
| 120~150 km | 1.22 | 55,939 | |
| 150~200 km | 1.31 | 13,791 | |
| 200~300 km | 1.29 | 5,006 | |

---

## 사용 방법

```python
def apply_distance_correction(osrm_km):
    if osrm_km <= 30: return osrm_km * 1.82
    elif osrm_km <= 60: return osrm_km * 1.52
    elif osrm_km <= 90: return osrm_km * 1.25
    elif osrm_km <= 120: return osrm_km * 1.25
    elif osrm_km <= 150: return osrm_km * 1.22
    elif osrm_km <= 200: return osrm_km * 1.31
    else: return osrm_km * 1.29
```

---

## 해석

- 가까울수록 보정비율이 높음 — OSRM 거리가 짧아서 시군구 내부 순회가 상대적으로 큰 비중
- 멀수록 보정비율 안정 (1.2~1.3) — RDC→시군구 이동 자체가 대부분
- 300km 이상은 데이터 부족(29건), 울릉군 등 특수 케이스로 제외

---

## 참고

- 배송 실적의 거리는 shipment 단위 (한 트럭의 전체 경로 거리)
- OSRM 거리는 RDC→시청 직행 기준
- 차이 = 시군구 내부 배송지 순회 + 복수 시군구 순회 포함 가능
