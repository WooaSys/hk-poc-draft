## 판단 기준

하나의 Region Code에 다른 광역도시의 배송지가 포함된 경우:
- 해당 시군구가 **자체 Region Code를 이미 가지고 있다면** → **오류 의심**
- 해당 시군구가 **자체 Region Code가 없다면** → 정상 (해당 RC가 유일한 소속)

## 권역 재배치 시 처리 방안

1. **무시**: 전체 3,320개 중 22건(0.7%)으로 영향 미미, 현행 유지
2. **자체 RC로 재배정**: 이탈 배송지를 해당 시군구의 자체 RC로 이동 (예: DEA의 KH11059 → IEA)
3. **고객사 확인 요청**: 실제 배송지 주소가 맞는지, RC가 맞는지 확인

## 참고

- 이 분석은 배송지 마스터의 `광역도시`, `시군구` 컬럼 기준
- 실제 배송지 주소(도로명 등)는 데이터에 미포함 → 시군구 정보 자체가 틀릴 가능성도 있음
- Region Code는 한국타이어 자체 권역 코드로, 반드시 행정구역과 일치하지 않음
