# 타이어 적재 기준표 — 보고서 요약

- **작성시간**: 2026-04-03

---

## 1. 왜 적재 기준표가 필요한가

### 목적

배송지(시군구)별 수요량을 정량화하기 위해서. "이 지역에 하루에 타이어가 얼마나 필요한가"를 트럭 단위로 환산해야 권역 재할당과 배차 시뮬레이션이 가능.

### 문제

고객사로부터 타이어의 정확한 **CBM(부피) 데이터를 제공받지 못함**. 타이어 크기(사이즈)에 따라 트럭에 실을 수 있는 양이 다른데, 이를 정량화할 기준이 없는 상태.

### 해결

배송 실적 데이터의 `Measurement by Material and Q'ty` 컬럼을 분석하여 **적재율 역산**. 이 컬럼은 "해당 shipment에서 타이어가 트럭 적재 용량의 몇 %를 차지했는가"를 나타냄.

---

## 2. 산출 근거

### 역산 방식

```
1본당 적재율(%) = Measurement ÷ Q'ty
```

- 동일 사이즈 × 동일 트럭 조합에서 1본당 적재율이 **고정값**임을 확인
- 변동계수(CV) **2.3%** — 거의 일정한 상수
- 즉, 타이어 사이즈와 트럭 톤수가 정해지면 1본이 차지하는 비율이 정해짐

### 결과

- **288개 타이어 사이즈** × **5개 트럭 톤수**(1T, 3.5T, 5T, 8T, 11T)별 적재율 산출
- 구분: PCR(승용차) 222개, LTR/TBR(경트럭/트럭버스) 66개

### 적재율 예시 (3.5T 기준)

| 사이즈 | 1본당 적재율 | 만적 수량 |
|---|---:|---:|
| 155/70R13 (소형) | 0.267% | 375본 |
| 225/45R18 (중형) | 0.400% | 250본 |
| 265/60R18 (대형) | 0.875% | 114본 |

→ 타이어가 클수록 1본이 차지하는 비율이 높고, 실을 수 있는 수량이 적음

---

## 3. 왜 3.5T와 5T 기준인가

### 트럭 톤수별 사용 현황 (자체 차량)

| 톤수 | 비중 | 역할 |
|---|---:|---|
| **3.5T** | **54.2%** | 주력 — 모든 RDC 공통 |
| **5T** | **31.7%** | 2순위 |
| 1T | 12.5% | 소량 배송 |
| 8T | 1.1% | 대량 |
| 11T | 0.5% | 대량 |

### 3.5T 선택 이유

1. 전체 자체 차량의 **54%**, 모든 RDC에서 최소 52%~100% 사용
2. 288개 사이즈 **전부** 3.5T 적재율 보유 (가장 완비된 기준)
3. RDC 간 수요 비교 시 **일관된 기준**으로 적합
4. 5T도 32%로 상당하나, 비교 기준 일관성을 위해 **3.5T 단일 사용**

### 5T 병행 산출

3.5T와 함께 5T 기준 수요도 산출하여 보유. 향후 정밀 시뮬레이션 시 활용 가능.

---

## 4. 대형 타이어와 8T/11T 트럭

### 대형 타이어 특성

LTR(경트럭)/TBR(트럭버스) 타이어 66개 사이즈 중 일부는 **8T, 11T 트럭에만 적재 가능**.

| 사이즈 예시 | 1T | 3.5T | 5T | 8T | 11T |
|---|---|---|---|---|---|
| 1100-20 (산업용) | - | - | - | - | ○ |
| 1200-20 (산업용) | - | - | - | - | ○ |
| 305/70R22.5 (TBR) | - | - | - | - | ○ |
| 180/650R15 (특수) | - | - | - | - | ○ |

### 영향

- 적재 기준표에 사이즈는 있으나 **3.5T/5T 값이 비어있는 9개 사이즈**
- 이 사이즈들은 주로 8T/11T 대형 트럭으로만 배송
- 3.5T 기준 수요 산출 시 **환산 불가** → 해당 건수는 미반영

### 적재 기준 매칭률

| 항목 | 값 |
|---|---|
| 전체 배송 건 | 354,960건 |
| 매칭 성공 | 328,547건 (92.6%) |
| 미매칭 | 26,413건 (7.4%) |

미매칭 원인:
1. **극소량 사이즈 (38개, 103건)**: 배송 실적이 10건 이하라 역산 불가, 수량 기준 0.02%
2. **대형 전용 사이즈 (9개)**: 3.5T/5T 적재율 없음, 대형 트럭 전용

→ **수요 산출에 실질적 영향 없음** (수량 기준 0.02%)

---

## 5. 수요 환산 방식

적재 기준표를 이용한 시군구별 수요 산출:

```
시군구 일일 수요 (3.5T 대수) = Σ(사이즈별 수량 × 3.5T 1본당%) ÷ 100
```

### 예시

어떤 시군구에 하루 배송:
- 225/45R18 80본 × 0.400% = 32.0%
- 245/45R18 60본 × 0.600% = 36.0%
- 합계: 68.0% → **3.5T 트럭 0.68대분**

이 값이 `daily_demand_3_5t`이며, 권역 재할당/배차 시뮬레이션/비용 산출의 기초 데이터.

---

## 6. 한계 및 참고

- CBM 데이터가 있으면 더 정확한 환산 가능
- 적재율은 **용적(부피) 기준**이며 중량 기준은 별도 고려 필요
- 3.5T 단일 기준이므로 실제 혼합 톤수 운영과 차이 있음
- 시나리오 간 **상대 비교 목적으로는 충분한 정확도**
