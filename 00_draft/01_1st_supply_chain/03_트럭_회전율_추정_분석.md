### 4-1. 검증 방법

차량 ID가 없으므로 **같은 날, 같은 RDC, 같은 트럭타입**에서 "이전 배송 도착 시각 < 다음 배송 상차 시작 시각"인 쌍을 찾아 2회전 간접 증거로 활용.

**신뢰도를 높이기 위한 필터링:**
- 갭 0분(다른 차량 가능성) 제외 → **갭 1~4시간**(복귀+대기+재상차에 현실적인 시간)만 채택

## 6. 분석 한계 및 유의사항

1. **차량 ID 없음** — 동일 차량의 복수 운행을 직접 추적 불가, 간접 추정만 가능
2. **복귀 시간 미포함** — 차량 점유시간은 편도(상차→도착)만 측정, 복귀(빈차 회차) 시간 별도 추정 필요
3. **Gate In/Out 데이터 0%** — RDC 체류(대기/상차/대기) 시간 직접 측정 불가
4. **Picking Start 데이터 8%** — 피킹 포함 전체 사이클은 표본이 작음(1,091건)
5. **Arrival 데이터 65%** — 35%는 도착 시각 미기록, 실제 회전율과 차이 가능
6. **제주RDC** — 유효 데이터 3건으로 분석 의미 없음

## 7. 핵심 인사이트

1. **2회전은 추정이 아닌 실데이터에서 확인된 사실** — 당일 완결 기준 22,927건, 3회전도 204건
2. **1T/3.5T는 1일 2회전이 현실적** — 차량 점유 중앙값 3시간 미만, 복귀 포함 6시간 이내
3. **전체 Shipment의 81%가 6시간 이내** — 대부분의 배송이 반나절 내 완료
4. **8T는 확실한 1회전형** — 장거리 대형 배송으로 15시간 점유, 당일 완결 2회전 증거 0건
5. **제천/계룡이 회전율 최적** — 배송 권역이 좁아 88/86% 6시간 이내
6. **평택은 물량 최대, 회전율 상대적 낮음** — 수도권 넓은 배송 범위로 77%
7. **회전 반영 시 필요 차량 36~45% 절감** — 272대 → 150~174대 (현실적 추정 160~180대)
8. **상차 시간은 병목이 아님** — 1T~5T는 4~5분으로 매우 효율적
9. **복귀~재상차 갭 중앙값 131분** — 배송 도착 후 약 2시간 내 다음 상차 시작이 일반적
