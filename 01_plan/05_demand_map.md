# 시군구별 수요 지도 시각화

- **작성시간**: 2026-04-02
- **주제**: 시군구별 차량 환산 수요를 지도에 시각화

## 요약

시군구별 3.5T/5T 환산 수요를 지도 위에 표시하여 수요 분포와 RDC 배정 현황을 직관적으로 파악한다.

## 배경

- 현재 RDC 현황 탭에 RDC별 권역 지도가 있음 (folium, 시군구 폴리곤)
- 수요 데이터(`sigungu_demand.csv`)가 준비됨
- 수요의 지리적 분포를 시각화하면 어디에 물량이 집중되는지, 현재 RDC 배정이 합리적인지 파악 가능

## 페이지 구성

자료 분석 페이지에 **"수요 분포"** 탭을 새로 추가.

### 상단: 안내 (expander)

- "산출 기준 안내" expander
  - 수요 산출 요약 (247개 시군구, 매칭률 92.6%)
  - 3.5T/5T 기준인 이유 (CBM 미보유, 적재율 기반 환산)
  - 실제와 차이가 있을 수 있음을 명시

### 중단: 지도

- **라디오 버튼**: 3.5T 기준 / 5T 기준 선택 → 지도에 즉시 반영
- **수요 히트맵 (choropleth)**: 시군구 폴리곤을 수요량에 따라 색상 농도로 표시
- **RDC 권역 표시**: 시군구별 담당 RDC 표시 (중복 권역은 더 많은 수요를 담당하는 RDC로 표시)
- **RDC 마커**: RDC 위치 별 마커
- **범례**: 수요 구간별 색상 + RDC 마커

## 입력 데이터

| 데이터 | 파일 |
|---|---|
| 시군구 수요 | `doc/wemeet/sigungu_demand.csv` |
| 시군구 shapefile | `raw_data/map/BND_SIGUNGU_PG.shp` |
| RDC 위치 | `app/db/rdc_locations.json` |
| 시군구 마스터 | `doc/wemeet/sigungu_master.csv` |

## 기술 스택

- Folium (choropleth + 마커)
- Streamlit

## 주의사항

- shapefile simplify(0.001) 적용 필요 (용량 문제)
- 제주 제외
- 수요 0인 시군구 처리 (색상 없음 또는 회색)
