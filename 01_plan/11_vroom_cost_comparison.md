# VROOM 배차 비용 비교 페이지

- **작성시간**: 2026-04-03
- **주제**: 현 상태 vs 재할당 시나리오의 VROOM 배차 결과 비용 비교

---

## 1. 목적

현 상태 권역의 배차 결과(3.5T 기준)와 재할당 시나리오의 배차 결과를 비용으로 비교한다. 권역 재할당이 실제 배차/운송 비용에 미치는 영향을 정량적으로 확인.

---

## 2. 전체 흐름

```
[1단계] 현 상태 VROOM 배차
  → 결과 저장 (한 번만 계산, 변하지 않음)

[2단계] 시나리오별 VROOM 배차
  → 권역 재할당 페이지에서 선택한 시나리오 기준

[3단계] 비용 비교
  → 현 상태 vs 시나리오 비용 차이
```

---

## 3. VROOM 배차 엔진

### 입력 구조

각 RDC별로 VROOM 요청을 구성:

- **vehicles**: 차량 수 = `ceil(Σ daily_demand_3_5t)`, capacity = 1.0 (만적 100%), start = RDC 좌표
- **jobs**: 해당 RDC에 배정된 시군구, demand = `daily_demand_3_5t`, location = 시군구 대표좌표

### VROOM 서버

- localhost:3000

### 출력

VROOM 결과에서 추출:
- 차량별 경로 (방문 시군구 순서)
- 차량별 총 이동거리/시간
- 차량별 적재율

### 거리 보정

VROOM이 산출하는 거리는 OSRM 기반(대표좌표 간)이므로, 실제 배송 거리와 차이 발생. `28_거리보정계수.md`의 거리 구간별 보정계수 적용:

| OSRM 거리 구간 | 보정계수 |
|---|---:|
| 0~30 km | 1.82 |
| 30~60 km | 1.52 |
| 60~90 km | 1.25 |
| 90~120 km | 1.25 |
| 120~150 km | 1.22 |
| 150~200 km | 1.31 |
| 200~300 km | 1.29 |

### 비용 산출

보정된 거리 → 단가표 거리구간 매핑 → RDC × 3.5T 단가 적용

단가표: `app/db/delivery_price_202511.csv` (2025년 11월)

---

## 4. 현 상태 배차 결과 (사전 계산)

현 상태 권역은 변하지 않으므로 한 번만 계산하고 저장:

- 파일: `app/db/baseline_vroom_result.json` (또는 csv)
- 내용: RDC별 차량 경로, 거리, 시간, 비용
- 페이지 로드 시 이 파일을 읽어서 사용

---

## 5. 시나리오별 배차

권역 재할당 페이지에서 선택한 시나리오의 배정 결과를 기반으로:
1. 시나리오별 RDC-시군구 배정 확인
2. VROOM 배차 실행
3. 거리 보정 + 비용 산출
4. 현 상태 결과와 비교

---

## 6. 페이지 구성

담당권역 재배치 그룹에 새 페이지 추가:

```
사이드바
├── 데이터 확인
│   └── 자료 분석
├── 담당권역 재배치
│   ├── 권역 재할당
│   └── 배차 비용 비교    ← 신규
└── 결과 비교
    └── (준비중)
```

### 페이지 레이아웃

```
┌──────────────────────────────────────────────────┐
│  배차 비용 비교                                    │
├──────────────────────────────────────────────────┤
│                                                  │
│  ■ 시나리오 선택                                  │
│  [시나리오 selectbox]  [실행 버튼]                  │
│                                                  │
│  ■ 비용 비교 요약                                  │
│  ┌──────────────────────────────────────────┐    │
│  │  현 상태 총 비용 vs 시나리오 총 비용        │    │
│  │  차이, 증감율                              │    │
│  └──────────────────────────────────────────┘    │
│                                                  │
│  ■ RDC별 비용 비교                                │
│  ┌──────────────────────────────────────────┐    │
│  │  그룹 막대 차트 (현 상태 vs 시나리오)        │    │
│  └──────────────────────────────────────────┘    │
│                                                  │
│  ■ 차량 경로 상세                                  │
│  ┌──────────────────────────────────────────┐    │
│  │  RDC 선택 → 차량별 경로, 적재율, 비용       │    │
│  └──────────────────────────────────────────┘    │
│                                                  │
└──────────────────────────────────────────────────┘
```
