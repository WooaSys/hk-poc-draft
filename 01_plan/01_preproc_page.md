# Streamlit App 프레임 구성

- **작성시간**: 2026-04-01 16:00 (수정: 16:30)
- **주제**: POC용 Streamlit 앱 전체 프레임 설계

## 요약

Streamlit 멀티페이지 앱의 사이드바 구조와 페이지 목록을 확정한다. 세부 페이지 내용은 각 페이지 작업 시 구체화.

## 앱 프레임

```
┌──────────────┬──────────────────────────────────────────────┐
│  사이드바      │                                              │
│              │              메인 컨텐츠 영역                  │
│  데이터 확인   │                                              │
│   ▸ 현재 상태  │                                              │
│   ▸ (추후)    │                                              │
│              │                                              │
│  최적화 실행   │                                              │
│   ▸ (추후)    │                                              │
│              │                                              │
│  결과 비교     │                                              │
│   ▸ (추후)    │                                              │
│              │                                              │
└──────────────┴──────────────────────────────────────────────┘
```

## 페이지 구성

| 그룹 | 페이지 | 상태 |
|---|---|---|
| 데이터 확인 | 현재 상태 | 1차 작업 대상 |
| 데이터 확인 | (추후 추가) | - |
| 최적화 실행 | (추후 추가) | - |
| 결과 비교 | (추후 추가) | - |
