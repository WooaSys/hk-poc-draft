# 권역 재할당 페이지 구성

- **작성시간**: 2026-04-02
- **주제**: 담당권역 재배치 > 권역 재할당 페이지 화면 설계

## 요약

Capacity Stress Test 엔진을 활용하여 평택RDC 물량 축소 시나리오를 실행하고, 결과를 시각화하는 페이지. 한 페이지에서 설정→실행→결과 확인.

## 페이지 구조

```
┌──────────────────────────────────────────────────────────────┐
│  권역 재할당                                                  │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ■ 파라미터 설정                                              │
│  ┌────────────────────────────────────────────────────────┐  │
│  │ 축소율 리스트: [5, 10, 15, 20]        (직접 수정 가능)  │  │
│  │ 수용 RDC 버퍼: [120] %                                 │  │
│  │ ☐ 중부RDC 포함                                         │  │
│  │                          [실행]                         │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  ■ 결과                                                      │
│                                                              │
│  1) 비용 증가 곡선                                            │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  축소율(%) vs 비용 증가율(%) 라인 차트                    │  │
│  │  → knee point 표시                                     │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  2) RDC별 물량 변화                                           │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  시나리오 선택 [S05 ▼]                                  │  │
│  │  RDC별 Before/After 막대 차트 + utilization             │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  3) 이동 시군구 지도                                          │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  선택된 시나리오의 이동 시군구 지도 (folium)              │  │
│  │  이동 전 RDC 색상 → 이동 후 RDC 색상                    │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  4) 이동 상세 테이블                                          │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  이동 시군구, from_rdc, to_rdc, demand, delta_time      │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  5) 시나리오 요약 테이블                                      │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  전체 시나리오 비교 (비용, 이동수, 증가율)                │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 파라미터 설정

| 파라미터 | UI | 기본값 | 비고 |
|---|---|---|---|
| 축소율 리스트 | text_input | `5, 10, 15, 20` | 콤마 구분, 0%는 자동 포함 |
| 수용 RDC 버퍼 | number_input | 120 | % 단위 |
| 중부RDC 포함 | checkbox | False | |

- 대상 RDC: 평택RDC 고정 (UI에 표시하지 않음)
- Solver: CP-SAT 사용 (주력 solver)

## 결과 시각화

### 1) 비용 증가 곡선

- X축: 축소율(%), Y축: 비용 증가율(%)
- Plotly 라인 차트
- 각 포인트에 값 표시
- knee point 구간 강조 가능

### 2) RDC별 물량 변화

- 시나리오 selectbox로 선택
- Before(baseline) / After 그룹 막대 차트
- utilization(%) 표시

### 3) 이동 시군구 지도

- 선택된 시나리오 기준
- folium 지도에 이동 시군구 강조
- 이동 전/후 RDC 색상 표시

### 4) 이동 상세 테이블

- 이동 시군구 목록
- from_rdc, to_rdc, demand, current_time, assigned_time, delta_time

### 5) 시나리오 요약 테이블

- 전체 시나리오 한눈에 비교
- 축소율, 총비용, 증가율, 이동 시군구 수, 이동 수요량

## 기술 스택

- Streamlit
- Plotly (차트)
- Folium (지도)
- stress_test 엔진 (CP-SAT)
