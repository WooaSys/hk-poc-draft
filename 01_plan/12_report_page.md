# 결과 보고서 페이지

- **작성시간**: 2026-04-03
- **주제**: 분석 결과를 한 화면으로 보고서 형식으로 구성 + PDF 다운로드

---

## 1. 목적

지금까지의 분석 결과(현황 분석 → 자연권역 → 권역 재할당 → 배차 비용)를 한 페이지에 보고서 형식으로 정리하여 고객/경영진에게 전달할 수 있도록 한다.

---

## 2. 위치

```
사이드바
├── 데이터 확인
│   └── 자료 분석
├── 담당권역 재배치
│   ├── 권역 재할당
│   └── 배차 비용 비교
└── 결과
    └── 보고서    ← 신규
```

---

## 3. 보고서 구성

### 1) 경영 요약

- 분석 목적: 평택RDC 편중 해소, 권역 재배정 검토
- 핵심 발견: 현재 권역은 자연권역과 94% 일치, 문제는 수요 편중
- 권장안: 평택 15% 축소 시 비용 증가 약 +3%, 현실적 분산 한계

### 2) 현황 분석

- RDC별 물량 비중 (원형 그래프)
- RDC별 배송지 분포 지도

### 3) 자연권역 분석

- 현재 vs 자연권역 비교 (94% 일치)
- 핵심 결론: 권역 왜곡이 아닌 수요 편중

### 4) 권역 재할당 결과

- 비용 증가 곡선 (이동시간/거리)
- 추천 시나리오의 RDC별 물량 변화
- 이동 시군구 지도 (Before/After)

### 5) 배차 비용 비교

- 현재 vs 시나리오 일평균 비용
- RDC별 비용 변화

### 6) 결론 및 제언

- 요약
- 제한사항 (3.5T 단일 기준, 시군구 대표좌표, 거리 보정 등)
- 다음 단계 제안

---

## 4. PDF 다운로드

- Streamlit 화면을 HTML로 렌더링 → PDF 변환
- 또는 reportlab/weasyprint 등으로 별도 PDF 생성
- 다운로드 버튼 제공

---

## 5. 기술 스택

- Streamlit
- Plotly (차트)
- PDF 생성: weasyprint 또는 pdfkit
